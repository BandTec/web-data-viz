Por que desse monitoramento?

As minhocas são seres vivos que como qualquer outro, estão sujeitas a mudanças na temperatura e na umidade no ambiente em que vivem. Manter uma temperatura dentro da faixa ideal de 20° a 25° é essencial para que as minhocas dentro da composteira vivam e se reproduzam de maneira saudavel. Manter a umidade dentro da faixa ideal de 70% a 80% também é um fator importante para manter as minhocas vivas. Esses 2 fatores são de suma importância para que composto final seja de extrema qualidade.

O que eu perco sem o monitoramento?

É uma questão complicada, mas não sem resposta e para ajudar com essa pergunta, nós da CompostECO desenvolvemos um simulador ficanceiro (calculadora) para você microprodutor. Para o funcionamento correto da calculadora, consideramos alguns pontos como verdade. O primeiro deles é que você utiliza 16g de adubo por m², o segundo ponto é que o preço do adubo que você utiliza é de 4,8 R$ cada 100 gramas (faixa ideal para o mercado), o terceiro é que uma vermicomposteira de 45 litros produz cerca de 12 quilos de adubo e o quarto é que com o monitoramento o valor do adubo aumenta em cerca de 50%.

Perguntas Frequentes - FAQ

Preciso ter conhecimento técnico para usar o sistema?

Não. O CompostEco foi desenvolvido para ser simples e intuitivo. A plataforma apresenta os dados de forma clara, permitindo que qualquer produtor acompanhe e tome decisões facilmente.

O sistema funciona offline?

Não totalmente. O sistema depende de conexão com a internet para enviar os dados do Arduino para a plataforma e permitir o monitoramento em tempo real.

Como recebo os alertas?

Os alertas são exibidos diretamente na dashboard quando a temperatura ou umidade saem da faixa ideal, permitindo uma ação rápida do usuário

O sistema toma alguma ação automaticamente?

Não. O CompostEco é um sistema de monitoramento passivo. Ele fornece dados e alertas, mas as decisões e ações são de responsabilidade do usuário.

Quais dados são monitorados?

Atualmente, o sistema monitora temperatura e umidade da composteira, que são os principais fatores para o bom funcionamento da vermicompostagem.

O sistema garante que as minhocas não morrerão?

Não. O sistema auxilia no monitoramento, mas não garante a sobrevivência das minhocas. A responsabilidade pelo manejo é do usuário.
